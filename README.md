
# OIDA

 **First install**
 
    npm install -g cordova
    cordova create oida
    cd oida
    cordova platform add browser
    cordova platform add ios
	cordova plugin add https://github.com/apache/cordova-plugin-camera.git

   Then clone this repo into `oida/www`
    
 **Run**

     cordova run browser
     cordova run ios
